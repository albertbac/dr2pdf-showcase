# Public Disclosure Policy

This repository is a public-safe technical case for dr2pdf.

## Allowed public material
- high-level product framing
- high-level architecture and workflow
- public-safe screenshots guidance
- public-safe positioning copy
- manual metrics placeholders

## Disallowed public material
- production source code
- raw operational content
- private infrastructure detail
- copied internal logic
- real screenshots or live records

## Publication rule
If a detail is not clearly safe, it stays out.
